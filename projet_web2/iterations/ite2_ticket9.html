Ticket 9 - Détection d'un compte déjà existant

L'objectif de ce ticket était de vérifier lors de l'inscription qu'un utilisateur ne puisse pas créer un compte avec un email déjà existant.

Modifications réalisées :
  • Ajout d'une vérification dans le fichier UtilisateurDao.class.php pour contrôler si l'email existe déjà dans la base de données.
  • Ajout d'une condition if/else dans le script d'inscription pour afficher un message d'erreur si l'email est déjà utilisé.
  • Modification de entete.html.php pour afficher les messages d’erreur de manière claire à l’utilisateur.
