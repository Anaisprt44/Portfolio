Consulter le consigne

Afin d’obtenir un classement correct des meilleurs restaurants, nous avons modifié le code de la fonction getTop4 dans le fichier RestoDAO.class.php.

Modifications réalisées :
  • Lignes 83 et 87 : modification de la requête SQL pour calculer correctement la moyenne des notes.
  • Dans le SELECT, remplacement de SUM(note) AS NotesCalculees par AVG(note) AS NotesMoyennes.
  • Ligne 87 : remplacement de ORDER BY NotesCalculees par ORDER BY NotesMoyennes pour trier correctement les restaurants.
