Mode WinForms


J’ai effectué un formulaire principal (FormMain) pour laisser à l’utilisateur le choix de la base de données à tester (en cliquant sur tâches en haut à gauche) :

L’utilisateur peut cliquer en haut sur tâches et pourra choisir la base :

On peut voir ici les différents choix que lui offre la base de données pour le test des différentes tables.

L’utilisateur peut tester toutes les URLs des logos des artisans et afficher leur code HTPS en cliquant sur « Tester toutes les URLS ».

Après le test, une boîte de dialogue s’affiche et les filtres s’activent.

On peut filtrer par code pour afficher les codes erreurs, les codes corrects ou tout.

Il est également possible de filtrer avec des combos box selon le nom des coopératives, entreprises, etc.

En sélectionnant une coopérative, la seconde liste déroulante, avec les entreprises, se met à jour automatiquement.

Cette mécanique est appliquée à plusieurs Forms, comme Interlocuteurs, Coopératives, etc.

Chaque Form est adapté selon les exigences, comme ici pour la table Interlocuteurs.

Un bilan est visible sur chaque Form, indiquant les logos corrects et incorrects après le test.
