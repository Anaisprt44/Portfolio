Projet Java

Projet Java (Avril 2024)

Nous avons fait courant avril 2024 (15/04/2024 au 19/04/2024) un projet Java par groupe de 2 qui était un projet de conception pour un service RH.

CONTEXE :
Nous devions pouvoir, par exemple, trier dans la liste déroulante par service les salariés, voir les informations des salariés, les modifier, les supprimer et la possibilité d’en créer des nouveaux.

Le contexte de notre projet était dans une entreprise, STESIO, où nous étions stagiaires. Nous étions l’ESN (entreprise de services du numérique) mandatée par INFOWARE pour développer son projet. On nous a demandé de programmer cette application qui permettra à l’entreprise de gérer ses salariés.

Compétences acquises

Gérer le patrimoine informatique : Vérifier les conditions de la continuité d’un service informatique à travers l’application de tri des services.

Répondre aux incidents et aux demandes d’assistance et d’évolution : Traiter des demandes concernant les applications, à travers le tri des services, consultation salarié, ….

Travailler en mode projet : Analyser les objectifs et les modalités d’organisation d’un projet pour créer notre site web.

Mettre à disposition des utilisateurs un service informatique : Déployer un service pour permettre à l’utilisateur de trier selon le nom du service, consulter fiche, ….

Organiser son développement professionnel : Mettre en place son environnement d’apprentissage personnel en cherchant comment développer cette application avec les Forms.

Conclusion Projet Java

Durant ce projet, j’ai pu développer mes compétences en travail de groupe et au niveau des interfaces graphiques avec Netbeans.
