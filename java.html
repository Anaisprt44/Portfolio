Projet Java (Avril 2024)

Bienvenue dans ce second projet de première année en BTS SIO

Nous avons fait courant avril 2024 (15/04/2024 au 19/04/2024) un projet Java par groupe de 2 qui était un projet de conception pour un service RH.


CONTEXTE :

Nous devions pouvoir, par exemple, trier dans la liste déroulante par service les salariés, voir les informations des salariés, les modifier, les supprimer et la possibilité d’en créer des nouveaux.

Le script SQL ainsi créé sera enfin exécuté dans le SGBD MariaDB pour créer les utilisateurs, les bases de données et leur accès.

Le contexte de notre projet était dans une entreprise, STESIO, où nous étions stagiaires. Nous étions l’ESN (entreprise de services du numérique) mandatée par INFOWARE pour développer son projet. On nous a demandé de programmer cette application qui permettra à l’entreprise de gérer ses salariés.
