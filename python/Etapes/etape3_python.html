Étape 3 - Création d’une base de données et d'un utilisateur

À cette étape, nous avons configuré une base de données et un utilisateur dans MariaDB en utilisant des commandes SQL.

Tout d’abord, nous avons créé une base de données portant votre nom d’utilisateur à l’aide de la commande SQL CREATE DATABASE. Ensuite, nous avons créé un utilisateur MariaDB avec votre nom d’utilisateur et lui avons attribué un mot de passe à l’aide de la commande CREATE USER.

Pour garantir que cet utilisateur dispose des autorisations nécessaires, nous lui avons donné les privilèges appropriés à l’aide de la commande SQL GRANT, lui permettant de créer, modifier et supprimer des tables et des colonnes, d’insérer, modifier et supprimer des données, ainsi que d’interroger sa base de données.

Enfin, nous avons testé ces commandes pour vérifier que l’utilisateur puisse bien effectuer les actions requises tout en garantissant qu’il ne puisse pas accéder à d’autres bases de données.
