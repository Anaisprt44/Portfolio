Étape 5a - Projet Python Création de mots de passe robustes

Pour renforcer la politique de mots de passe chez STESIO, nous avons utilisé un tableur pour générer des mots de passe répondant à des critères de sécurité. Chaque utilisateur extrait du fichier usersToulouse.csv a reçu un mot de passe respectant les critères, comme une combinaison aléatoire de lettres majuscules et minuscules ainsi que de chiffres. Nous avons exporté ces mots de passe dans un fichier CSV, prêts à être utilisés.
