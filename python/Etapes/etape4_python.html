Étape 4 - Extraction des utilisateurs de l'annuaire AD

À l’étape 04, nous avons récupéré des données sur certains utilisateurs de l’annuaire Active Directory d’un serveur de domaine Windows. Après, nous avons utilisé le logiciel LDAPBrowser pour accéder à l’annuaire et extraire les informations nécessaires. Après avoir configuré LDAPBrowser pour se connecter au serveur CDSTESIO, nous avons extrait les données des utilisateurs de l’unité d’organisation (OU) Toulouse. Ces données ont été exportées au format CSV, en spécifiant les attributs à inclure (nom et prénom) et en définissant le format de sortie approprié. Ensuite, nous avons transféré le fichier CSV contenant les informations des utilisateurs de votre VM Windows 10 vers votre VM de développement via notre espace réseau personnel. Ce processus nous a permis de collecter les données requises pour créer les comptes utilisateurs et les bases de données dans MariaDB.
