Étape 5 - Projet Java

La Mission 5 consiste à ajouter une fonctionnalité à notre site web où les utilisateurs peuvent sélectionner une catégorie de livres à partir d’une liste déroulante. Les catégories sont présentées dans la liste déroulante par ordre alphabétique, ce qui nous était imposé. Lorsque l’utilisateur clique sur un bouton « Afficher les livres », une liste des différentes catégories est affichée et lorsque l’on sélectionne l’une d’entre elles, les livres de celle-ci s’affichent, comprenant leur IBAN, titre et leur auteur.

Comme on peut le voir ci-dessous, nous avons ajouté dans une page réservation la possibilité au client de sélectionner une catégorie et d’afficher uniquement les livres de cette catégorie :

Lorsque l’on sélectionne par exemple la catégorie « Roman », les livres de cette catégorie s’affichent.
