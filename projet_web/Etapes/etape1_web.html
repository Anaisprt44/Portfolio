Etape 1 – Projet Web

Consigne Etape 1

Dans cette étape, on nous a expliqué que nous devions durant le projet réaliser plusieurs pages comme :

    Une page d’accueil avec :
    • Un mot de bienvenue
    • Un widget Météo France
    Une page avec une galerie photos comprenant :
    • La bibliothèque
    • Des livres
    • L’équipe
    Une page libre avec le contenu de notre choix.
    Une page « contact » avec :
    • L’adresse
    • Le téléphone
    • L’adresse mail
    • Un widget Google Maps
    Un menu de navigation pour faciliter la navigation sur le site.

Contraintes du projet :

  • Le site devait être codé en HTML5 et CSS3.
  • Utiliser une feuille de style CSS unique pour toutes les pages (pas de CSS dans le HTML).
