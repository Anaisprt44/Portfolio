Ticket 1 - Authentification utilisateur

Programmes :

Dans ce ticket, nous avons mis en place le système d’authentification des utilisateurs pour l’application R3st0.fr.

Tout d’abord, une interface de connexion a été développée afin de permettre aux modérateurs de saisir leur identifiant et leur mot de passe.

Ensuite, côté backend, une méthode de vérification a été créée dans la couche DAO afin de comparer les identifiants saisis avec ceux présents dans la base de données.

Cette vérification permet de valider ou non l’accès à l’application.

En cas de réussite, l’utilisateur est redirigé vers l’écran principal de gestion des avis. Dans le cas contraire, un message d’erreur est affiché afin d’indiquer une authentification incorrecte.

Enfin, une gestion de session a été ajoutée afin de maintenir l’utilisateur connecté pendant son utilisation de l’application.

Test fonctionnel :

Objectif : Vérifier que l’authentification fonctionne correctement.

1. Lancer l’application.
2. Saisir un identifiant et un mot de passe valides.
3. Vérifier l’accès à l’écran principal de modération.

Cas d’erreur :
- Identifiants incorrects → message d’erreur affiché.
- Utilisateur inexistant → accès refusé.

Tâches réalisées :
  • Création de l’interface de connexion utilisateur.
  • Implémentation de la vérification des identifiants en base de données.
  • Gestion des erreurs d’authentification.
  • Mise en place d’une session utilisateur après connexion.
  • Redirection vers l’interface principale de l’application.
