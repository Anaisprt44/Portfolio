Ticket 2 - Affichage de la liste des avis

Accéder à la branche GitLab
Programmes :

Création dans CritiqueDAO de plusieurs méthodes pour récupérer les avis depuis la base de données :

  • getAll() : récupère toutes les critiques de la table critiquer et retourne une liste d’objets Critique.
  • getByRestoId() : récupère les avis d’un restaurant selon son identifiant, avec gestion des valeurs NULL.
  • getAllWithRestaurantNom() : récupère les critiques avec le nom du restaurant via jointure SQL.
  • getAllWithRestoAndUser() : récupère les critiques avec le nom du restaurant et le pseudo de l’utilisateur via jointures.

Ces méthodes utilisent des requêtes SQL avec PreparedStatement, parcourent les ResultSet et construisent des objets Critique.

Ajout d’un champ booléen (masqué) dans la table critiquer permettant de gérer l’affichage des avis (par défaut : false).

Dans ListeAvisPanel, appel de la méthode chargerAvis() dans le constructeur.

La méthode chargerAvis() récupère les avis via le DAO puis alimente le JTable : elle vide d’abord le tableau, puis ajoute chaque critique (date, utilisateur, restaurant, commentaire, statut affiché/masqué).

Test fonctionnel :

  • Redirection vers l’écran principal après login
  • Affichage correct des critiques (date, auteur, restaurant, commentaire, statut)
