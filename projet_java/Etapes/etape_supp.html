Étape supplémentaire - Projet Java

Cette étape supplémentaire consiste à consulter une vue d'ensemble de tous les salariés, puis à sélectionner un salarié en particulier. Pour cet exemple, nous avons choisi le salarié **S01 (Bernard Céline)**. Une fois que le salarié est sélectionné, une liste déroulante est affichée, contenant toutes les formations disponibles dans le système.

Dans cet exemple, nous avons sélectionné la formation intitulée **"Administration d'un réseau local sous Linux"**. Une fois la formation choisie, il suffit de cliquer sur le bouton "Ajouter" pour l'associer au salarié sélectionné. Cette action permet d'ajouter la formation dans le profil du salarié.

Après l'ajout de la formation, vous pouvez voir qu'elle a bien été ajoutée à la liste des formations du salarié. Cela permet de garantir que le salarié possède bien cette compétence.

Il est également possible de vérifier l'ajout dans la base de données SQL. En accédant à la table associée aux formations et en filtrant par le numéro du salarié, vous pourrez confirmer que la formation a bien été enregistrée.

À la fin de ce processus, en suivant les liens entre les tables et le numéro du salarié, nous pouvons confirmer que la formation a bien été ajoutée au profil du salarié dans la base de données. Cette vérification garantit la cohérence des données et l'intégrité du système de gestion des formations.
