Etape 7 – Projet Intranet

Consigne Etape 7 Voir compte Rendue

Objectifs

Ce projet consiste à créer un formulaire HTML pour la création d'utilisateurs dans un espace d'administration, et à intégrer ces utilisateurs dans une base de données. On devait par exemple faire ces tâches :

  • 1. Créez un formulaire HTML pour la création d’un utilisateur depuis l’espace d’administration
  • 2. Ajoutez les groupes 'Administrateur', 'Ressources Humaines', 'Logistique' et 'Informatique' manuellement dans votre base de données
  • 3. Ajoutez le code PHP pour récupérer les groupes dans la base de données et les afficher dans la liste du formulaire
  • 4. Utilisez PHP pour récupérer les valeurs du formulaire via la méthode POST pour créer l’utilisateur dans la base de données

    • Question 1 - Créez un formulaire HTML pour la création d’un utilisateur depuis l’espace d’administration

    Nous avons conçu un formulaire permettant à l’administrateur de créer un utilisateur. Ce formulaire inclut des champs pour le nom, prénom, login, et mot de passe.

    Le formulaire utilise la méthode POST et permet également l'upload de fichiers (comme une photo de profil) grâce à l'attribut `enctype="multipart/form-data"`.


    Question 2 - Ajoutez les groupes 'Administrateur', 'Ressources Humaines', 'Logistique' et 'Informatique' manuellement dans votre base de données

    Nous avons inséré ces groupes dans la base de données à l'aide de la commande SQL suivante :

    INSERT INTO groupes (codeGrp, libelle) VALUES ('1', 'Administrateur'), ('2', 'Ressources Humaines'), ('3', 'Logistique'), ('4', 'Informatique');

    Les groupes ont bien été ajoutés à la base de données.


    Question 3 - Ajoutez le code PHP pour récupérer les groupes dans la base de données et les afficher dans la liste du formulaire

    Le code PHP permet de récupérer les groupes depuis la base de données et de les afficher dans une liste déroulante dans le formulaire. Si aucun groupe n'est trouvé, un message est affiché.

    La requête SQL SELECT est utilisée pour récupérer les groupes et les afficher dynamiquement dans des éléments


    Les données du formulaire sont récupérées via la méthode POST. Un hachage du mot de passe est effectué pour sécuriser les données avant leur insertion dans la base de données.

    Une requête SQL préparée insère les informations dans la table des utilisateurs. Si l’insertion est réussie, un message de succès est affiché. En cas d'erreur, un message d'erreur est montré.
