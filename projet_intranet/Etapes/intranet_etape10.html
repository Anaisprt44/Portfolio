Fonction PHP pour récupérer les notifications

  • Vérification de la connexion utilisateur via la session.
  • Préparation d'une requête SQL pour récupérer les notifications de l'utilisateur triées par date décroissante.
  • Renvoi des notifications sous format JSON ou affichage d'un message si aucune notification n'est trouvée.
  • Gestion des erreurs SQL avec renvoi d'un message détaillé en cas de problème.
