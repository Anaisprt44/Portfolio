Ensuite, nous devions créer une base de données dans MariaDB avec le nom "corpany_project_intranet". Tout d’abord dans le script SQL de la base de données, on supprime la base si elle existe déjà pour éviter les erreurs de type : la base de données est déjà existante. Puis on crée la base de données avec la commande « CREATE DATABASE nom_table ». On donne le nom demandé, c’est-à-dire "corpany_project_intranet".
