Projet WEB

Projet Web (Année 2024)


Bienvenue dans ce projet de première année en BTS SIO. Nous devions travailler par 3 sur un projet WEB et créer un site web pour ue bibliothèque associative :

CONTEXE :

Dans ce projet, nous étions responsables d’une bibliothèque associative et nous devions mettre en place un site internet de gestion des livres à destination des habitants de votre ville. Pour le mener à bien, notre projet était organisé en différentes missions :

En voir plus. Nous n’avions pas de cahier des charges donné pour ce projet. Pour réussir, nous devions travailler en groupe de 3, et nous avions un créneau horaire de 4 heures pour travailler sur ces différentes missions sur la période de janvier à avril 2024.

Lien vers GIT

TECHNOLOGIES UTILISÉES :

Langages utilisés : HTML/CSS, PHP, JavaScript et SQL

Logiciel de développement : NetBeans

Serveur Web : Wamp


Compétences Tableau de Compétences :

Gérer le patrimoine informatique

  • Recenser et identifier les données de la bibliothèque et les utiliser pour le site

    • (Recenser et identifier les ressources numériques)
  • Enresgistrer les livres empruntés par les utilisateurs

    • (Gérer des sauvegardes)
  • Vérification du « niveau » de l’utilisateur (si c’est un client/admin) pour accéder aux différents services du site,

    • (Mettre en place et vérifier les niveaux d’habilitation associés à un service)

Répondre aux incidents et aux demandes d’assistance et d’évolution

  • Collecter les données, les organiser et traiter les demandes des utilisateurs pour les emprunts des livres

    • (Collecter, suivre et orienter des demandes)

Développer la présence en ligne de l’organisation

  • Faire évoluer le site Web exploitant les données de la bibliothèque et des adhérents

    • (Participer à l’évolution d’un site Web exploitant les données de l’organisation.)

Organiser son développement professionnel

  • Développer son projet professionnel en apprenant davantage sur JS/PHP

    • (Développer son projet professionnel)

Conclusion projet :

Ce projet qui s’est déroulé sur plusieurs semaines m’a permis de créer en équipe un site web. De m’améliorer dans le langage PHP, et également en JavaScript. J’ai pu également appliquer les cours pour éviter les injections SQL dans le site que nous avons créé.
