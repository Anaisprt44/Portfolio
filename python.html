Bienvenue dans ce premier projet de première année en BTS SIO

Bienvenue dans ce premier projet de première année en BTS SIO. Nous devions travailler en binôme sur un projet Python de gestion des utilisateurs.

L'objectif était de créer un script Python permettant de générer des comptes utilisateurs avec leur mot de passe et leurs accès dans un fichier CSV à partir d'une base de données.

Nous avons appris comment générer des comptes utilisateurs avec des mots de passe plus ou moins longs (avec des caractères spéciaux etc...).

Le script SQL ainsi créé sera enfin exécuté dans le SGBD MariaDB pour créer les utilisateurs, les bases de données et leur accès.
